Testes de Fluxo de Reserva - BlazeDemo

Este projeto valida o fluxo de reserva de passagens no site BlazeDemo, utilizando duas abordagens: teste manual e teste automatizado com Playwright.

- Teste Manual: execução do fluxo completo de reserva simulando o usuário, incluindo seleção de origem/destino, escolha do voo, preenchimento dos dados e validação da confirmação da compra e valor final.
- Teste Automatizado: implementação do mesmo fluxo utilizando Playwright, com validações de elementos, navegação entre páginas (URL) e confirmação de sucesso da compra.
