BlazeDemo – Testes de Fluxo de Reserva (Versão Básica)

Este projeto valida o fluxo de reserva de passagens no site BlazeDemo, utilizando duas abordagens: teste manual e automação com Playwright.

📌 Abordagens utilizadas

Teste Manual: execução do fluxo completo simulando o usuário, incluindo seleção de origem/destino, escolha do voo, preenchimento dos dados e validação da confirmação da compra e valor final.
Teste Automatizado: implementação do mesmo fluxo com Playwright, incluindo validações de elementos, navegação entre páginas (URL) e verificação de sucesso da compra.

📚 Objetivo
Servir como base para entendimento do fluxo E2E e introdução à automação de testes, sendo posteriormente evoluído para uma versão mais estruturada e escalável.
